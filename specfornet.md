# Lemon Subtitle Studio - .NET 技术规格文档

## 1. 需求分析

### 1.1 业务背景
基于现有 Rust + Tauri 项目，迁移到 .NET 技术栈，实现视频字幕提取、识别和翻译功能。保留原有的界面操作结构和用户体验。

### 1.2 核心功能需求

| 功能模块 | 功能描述 | 对应原组件 |
|---------|---------|-----------|
| **音频提取** | 从视频文件中提取音频轨道 | `AudioExtractor.tsx` |
| **语音识别** | 使用 Whisper 模型将音频转换为字幕文本 | `ASRStudio.tsx` |
| **字幕翻译** | 使用 ONNX 翻译模型进行中英互译 | `TranslationDesk.tsx` |
| **批量处理** | 支持多个文件的批量处理，带进度显示和日志 | `BatchWorkspace.tsx` |
| **模型管理** | 下载、删除、配置 Whisper 和翻译模型 | `Settings.tsx` |
| **历史记录** | 记录处理历史，支持查询和管理 | `commands.rs` |

### 1.3 界面操作结构

#### 1.3.1 字幕提取页面 (Video to Subtitle)
- **操作流程**：导入文件 → 预览视频 → 开始提取 → 编辑字幕 → 导出字幕
- **完成后可双击预览视频**，完成之前不需要预览
- **编辑字幕**：点击编辑按钮直接跳转到字幕编辑页面，同时将视频和字幕文件路径传递过去
- **核心交互**：文件选择、进度显示、字幕列表编辑、时间轴跳转

#### 1.3.2 视频转音频页面 (Video to Audio)
- **操作流程**：导入文件 → 配置参数 → 开始提取 → 导出结果
- **核心交互**：文件选择、格式配置、进度显示

#### 1.3.3 音频转字幕页面 (Audio to Subtitle)
- **操作流程**：导入音频 → 开始识别 → 导出字幕
- **不需要预览波形**，不需要编辑字幕
- **编辑字幕**：点击编辑按钮直接跳转到字幕编辑页面，同时将音频和字幕文件路径传递过去
- **核心交互**：文件选择、进度显示

#### 1.3.4 字幕翻译页面 (Subtitle Translation)
- **操作流程**：导入字幕 → 选择语言 → 开始翻译 → 编辑译文 → 导出结果
- **核心交互**：双语对比编辑、翻译进度

#### 1.3.5 字幕编辑页面 (Subtitle Editor)
- **操作流程**：选择视频/音频和字幕 → 预览 → 边看边编辑 → 导出字幕
- **预览区域**：根据导入类型自动切换（视频界面或音频波形）
- **核心交互**：时间轴同步、字幕时间调整、波形/视频预览切换

#### 1.3.6 设置页面 (Settings)
- **操作流程**：查看模型列表 → 下载/删除模型 → 配置默认模型 → 配置默认保存位置
- **核心交互**：模型管理、设置保存、默认保存位置配置

## 2. 技术选型

### 2.1 技术栈

| 分类 | 技术 | 版本 | 选型理由 |
|------|------|------|---------|
| **语言** | C# | 12.0 | 现代语法，性能优秀，生态成熟 |
| **框架** | .NET 8 | 8.0 | LTS 版本，性能提升，原生 AOT 支持 |
| **UI** | WPF | 8.0 | Windows 原生 UI，成熟稳定，生态丰富 |
| **语音识别** | Whisper.net | 1.5+ | Whisper 的官方 .NET 封装 |
| **推理引擎** | ONNX Runtime | 1.16+ | 跨平台推理，支持 DirectML/CUDA |
| **GPU 加速** | DirectML + CUDA | - | 双后端支持，自动检测最佳方案 |
| **视频处理** | Xabe.FFmpeg | 5.0+ | 成熟的 FFmpeg .NET 封装 |
| **数据库** | SQLite | 3.45+ | 轻量级，无需额外服务 |
| **MVVM** | Prism | 8.0+ | 成熟的 WPF MVVM 框架 |

### 2.2 关键技术决策

| 决策项 | 选择 | 说明 |
|--------|------|------|
| **UI 框架** | WPF | 成熟稳定，生态丰富，适合 Windows 原生应用 |
| **GPU 加速** | DirectML + CUDA | 自动检测可用方案，优先 CUDA，回退 DirectML/CPU |
| **模型分发** | 内置基础模型 + 按需下载 | 内置 tiny/base 模型，large 等大模型按需下载 |
| **打包格式** | MSIX + 自包含 EXE | MSIX 用于 Store，EXE 用于直接分发 |
| **跨平台** | Windows only | 聚焦 Windows 平台，优化最佳体验 |
| **最低版本** | Windows 10 1809+ | DirectML 原生支持的最低版本 |

### 2.3 架构风格
- **分层架构**：UI 层 → 业务层 → 基础设施层
- **MVVM 模式**：使用 Prism 实现视图与逻辑分离
- **模块化设计**：按功能模块划分独立组件

### 6.2 Entity Framework Core 配置

```csharp
public class AppDbContext : DbContext
{
    public DbSet<HistoryRecord> HistoryRecords { get; set; }
    public DbSet<ModelSetting> ModelSettings { get; set; }

    protected override void OnConfiguring(DbContextOptionsBuilder options)
    {
        string dbPath = Path.Combine(
            Environment.GetFolderPath(Environment.SpecialFolder.ApplicationData),
            "LemonSubtitleStudio",
            "app.db"
        );
        options.UseSqlite($"Data Source={dbPath}");
    }

    protected override void OnModelCreating(ModelBuilder modelBuilder)
    {
        modelBuilder.Entity<ModelSetting>()
            .HasIndex(s => s.SettingKey)
            .IsUnique();
    }
}

public class ModelSetting
{
    public int Id { get; set; }
    public string SettingKey { get; set; } = string.Empty;
    public string SettingValue { get; set; } = string.Empty;
}
```

| **模型列表** | 显示可用模型及状态 | 下载、设为默认、删除操作 |
| **模型存储路径** | 配置模型文件存放位置 | 输入框 + 浏览按钮 |
| **默认输出目录** | 配置所有功能页面的默认保存位置 | 输入框 + 浏览按钮，各功能页面的输出目录选择初始值取自此配置 |
| **字幕命名规则** | 设置输出字幕文件的命名格式 | 单选按钮切换，支持自定义格式 |
| **恢复默认设置** | 重置所有设置为默认值 | 点击弹出确认对话框 |

**关键交互说明**：
- **默认输出目录**：所有功能页面（视频转字幕、音频转字幕、字幕翻译等）的输出目录选择器初始值取自此配置，用户可在各页面按需覆盖
- **模型存储路径**：独立于输出目录，专门用于存放 Whisper 模型文件
| **命名规则选择** | 字幕文件命名方式 | 按钮组切换 |
| **自定义命名格式** | 自定义输出文件名 | 输入框 |

### 7.4 交互规范

#### 7.4.1 通用交互

#### 7.4.2 任务状态与重试机制

| 任务状态 | 视觉表现 | 可用操作 |

| 失败 | 红色 X 标记 | **重试**、删除 |

#### 7.4.3 多阶段进度展示

视频转字幕任务分阶段显示进度：

| 阶段 | 说明 |
|------|------|

| 字幕识别 | 进度条 2（橙色） |
| 字幕翻译 | 进度条 3（紫色，可选） |

#### 7.4.4 性能监控显示

主内容区右上角显示实时性能信息：

```
┌────────────────────────────────────────────────────────┐
│  视频转字幕  [X个文件]   [内存: 256MB] [CPU: 12%] [导出全部] [开始处理] │
└────────────────────────────────────────────────────────┘
```

| 指标 | 显示格式 | 更新频率 |
|------|---------|---------|
| 内存使用 | `内存: XXXMB` | 1秒 |
| CPU 使用率 | `CPU: XX%` | 1秒 |

#### 7.4.5 文件覆盖处理

当输出文件已存在时，弹出对话框：

```
┌────────────────────────────────────────┐
│           文件已存在                     │
├────────────────────────────────────────┤
│  输出目录中已存在以下文件：              │
│                                        │
│  □ 视频1.srt                           │
│  □ 视频2.srt                           │
│                                        │
│  [ ] 对所有文件执行相同操作              │
│                                        │
│  [覆盖] [跳过] [重命名] [取消]          │
└────────────────────────────────────────┘
```

| 选项 | 行为 |
|------|------|
| 覆盖 | 直接覆盖已有文件 |
| 跳过 | 跳过该文件，继续处理下一个 |
| 重命名 | 自动在文件名后添加数字后缀（如 `视频1_1.srt`） |
| 取消 | 取消所有文件的处理 |

#### 7.4.6 可折叠面板交互

右侧面板各区域支持折叠/展开：

```
┌─────────────────────────────────────────┐
│ ▼ 视频预览          [高度: 固定] [折叠] │
│ ▼ 字幕列表          [高度: 自适应] [折叠] │
│ ▶ 任务进度          [当前隐藏]          │
│ ▶ 处理日志          [当前隐藏]          │
└─────────────────────────────────────────┘
```

| 状态 | 交互 |
|------|------|
| 展开 | 点击标题栏或展开按钮 |
| 折叠 | 点击标题栏或折叠按钮 |
| 固定高度 | 某些区域（如视频预览）可折叠但不可完全折叠到0 |

#### 7.4.7 确认对话框

不可逆操作弹出确认对话框：

| 操作 | 确认内容 |
|------|---------|
| 删除文件 | "确定要从队列中删除此文件吗？" |
| 清空队列 | "确定要清空所有文件吗？此操作不可撤销。" |
| 取消任务 | "确定要取消当前正在处理的任务吗？" |

#### 7.4.8 多语言界面支持

软件界面支持多语言切换：

| 组件 | 说明 |
|------|------|
| 语言选择器 | 侧边栏底部，中/EN/日本語 等切换 |
| 资源文件 | 使用 `.resx` 资源文件管理界面文本 |
| 动态切换 | 无需重启，实时切换界面语言 |

#### 7.4.9 队列状态持久化

| 状态 | 保存时机 | 恢复方式 |
|------|---------|---------|
| 队列文件列表 | 每添加/删除文件时 | 启动时自动恢复 |
| 任务进度 | 实时保存（每5秒） | 启动时可选择恢复或重新开始 |
| 用户配置 | 立即保存 | 启动时自动加载 |

#### 7.4.10 临时文件清理

| 时机 | 清理内容 |
|------|---------|
| 任务完成后 | 自动删除该任务的临时音频/中间文件 |
| 应用关闭时 | 清理所有未完成的临时文件 |
| 例外 | 用户可设置保留临时文件供调试 |
